# TTNmon-Gateway-Stats
A packet collector and forwarder to create RX stats of a gateway for TTNmon.meis.space

## Requirements
- Semtech UDP Forwarder
- Python3
- python-requests

## Setup
``apt install python3 python3-requests``
``python3 ttnmon_forwarder.py``
